# 🧪 Testplan: AI-Driven Editing — End-to-End Lazynext AI Agent Pipeline

> **Feature**: `18` — AI-Driven Editing

---

## Acceptance criteria

| # | Criterion | Verified by |
|---|---|---|
| AC1 | User types "delete filler words from track 1" → timeline visibly changes (filler clips removed/shortened) | E2E test (E.2) |
| AC2 | User types "transcribe this video" → captions appear on caption track | E2E test (E.2) |
| AC3 | User types "auto-reframe to vertical" → project resolution changes to 9:16; crop keyframes appear | E2E test (E.2) |
| AC4 | AI chat shows step-by-step progress (not a single final result) | UI test |
| AC5 | When a microservice is down, the chat shows a clear error (not silent no-op) | Unit test |
| AC6 | CRDT patches generated by the orchestrator are accepted by the WASM engine and produce visible changes | Round-trip test (E.1) |
| AC7 | `bun run test`, `cargo test`, `bun run typecheck` all pass (0 failures) | CI check |
| AC8 | All Tier 1 tools return success when their microservice is available; all return structured errors when unavailable | Integration test |

## Test cases

### TC1: CRDT patch round-trip (unit — Rust)
**Setup**: Instantiate WASM engine with empty timeline.  
**Input**: A `transcribe`-style CRDT patch:
```json
{"op": "add", "path": "/tracks/caption_track/clips/caption_1", "value": {"type": "TextLayer", "text": "Hello", "start": 0, "end": 5}}
```
**Expected**: `engine.applyOperation(patch)` returns `true`. `engine.getEntityGraph()` includes a clip on `caption_track` with `text: "Hello"`.  
**Status**: [ ] Pass / [ ] Fail

### TC2: Transcribe tool → real endpoint (integration)
**Setup**: Pre-processing `:8000` running with a sample video.  
**Input**: `executeToolCall("transcribe", {video_id: "test_video"})`  
**Expected**: Returns `{success: true}` with `crdt_patches` containing TextLayer clips.  
**Status**: [ ] Pass / [ ] Fail

### TC3: Filler removal → timeline mutation (integrated)
**Setup**: Timeline with audio clip containing filler words.  
**Input**: `executeToolCall("clean_audio", {video_id: "test_video"})`  
**Expected**: Returns `{success: true}` with patches that remove or trim filler regions.  
**Status**: [ ] Pass / [ ] Fail

### TC4: Missing microservice → graceful error (unit)
**Setup**: Pre-processing not running.  
**Input**: `executeToolCall("transcribe", {video_id: "test_video"})`  
**Expected**: Returns `{success: false, error: <connection error>}`. Orchestrator retries 2x, then the SSE stream emits an error event. AI chat displays the error.  
**Status**: [ ] Pass / [ ] Fail

### TC5: Unknown tool → failure (unit)
**Setup**: Any.  
**Input**: `executeToolCall("nonexistent_tool", {})`  
**Expected**: Returns `{success: false, error: "Unknown tool: nonexistent_tool"}`.  
**Status**: [ ] Pass / [ ] Fail

### TC6: `apply_color_grade` → wired (unit)
**Setup**: Pre-processing running.  
**Input**: `executeToolCall("apply_color_grade", {clip_id: "test_video", preset: "cinematic"})`  
**Expected**: Returns `{success: true}` with CRDT patches for color effect. (Currently fails — this test validates the Phase B.1 fix.)  
**Status**: [ ] Pass / [ ] Fail

### TC7: SSE streaming (unit)
**Setup**: `executePlan()` with 3-step plan.  
**Input**: Subscribe to SSE stream.  
**Expected**: 6+ events emitted: `step:start` × 3, `step:result` × 3, `plan:complete`. Each event contains the tool name and result.  
**Status**: [ ] Pass / [ ] Fail

### TC8: E2E — AI chat → timeline (Playwright)
**Setup**: Web app running with ai-agents + pre-processing. Sample video loaded on timeline.  
**Steps**:
1. Open AI chat panel
2. Type: "remove filler words"
3. Press Enter
**Expected**:
- Chat shows progress messages ("Analyzing audio...", "Found 5 filler words", "Removing... ✓")
- Timeline clips visibly change (filler regions removed)
- Undo button appears; clicking it restores original timeline
**Status**: [ ] Pass / [ ] Fail
