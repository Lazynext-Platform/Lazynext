# Internationalization (i18n) Pipeline

Lazynext supports **18 languages**, **190 countries**, and **100+ currencies** across all 7 platform formats.

## Adding a New Language

### Add locale file in each platform:

| Platform | Directory | Format | Example |
|----------|-----------|--------|---------|
| API Gateway | `rust/api-gateway/locales/{lang}.yml` | YAML | `fr.yml` |
| CLI | `rust/cli/locales/{lang}.yml` | YAML | `fr.yml` |
| Desktop | `apps/desktop/locales/{lang}.yml` | YAML | `fr.yml` |
| Web | `apps/web/messages/{lang}.json` | next-intl JSON | `fr.json` |
| Mobile | `apps/mobile/locales/{lang}.json` | i18next JSON | `fr.json` |
| Extension | `apps/browser-extension/public/_locales/{lang}/messages.json` | Chrome i18n | `fr/messages.json` |
| MCP Server | `services/mcp-server/src/i18n.ts` | Inline TS | Add to `messages` object |

### Register in locale lists:

| Platform | File | Variable |
|----------|------|----------|
| Web | `apps/web/src/i18n.ts` | `ALL_LOCALES` array |
| Web | `apps/web/src/proxy.ts` | `locales` array (x2) |
| Web | `apps/web/src/components/locale-selector.tsx` | `SUPPORTED_LOCALES` array |

## Adding a New Currency

### Rust:
```rust
// rust/crates/international/src/currencies.rs
Currency { code: "XYZ", numeric: 999, name: "Example", symbol: "¤", decimals: 2, symbol_before: true },
```

### JS (auto-supported via Intl.NumberFormat):
No changes needed - the browser already knows all ISO 4217 currencies.

### Add to UI selector:
```typescript
// apps/web/src/components/locale-selector.tsx → COMMON_CURRENCIES
{ code: "XYZ", name: "Example Currency", symbol: "¤" },
```

## Adding a New Country

### Rust:
```rust
// rust/crates/international/src/countries.rs
Country { code: "XX", name: "Example", native_name: "Example", continent: "...", default_locale: "...", default_currency: "...", calling_code: "+...", is_eu: false },
```

### JS CountrySelector:
```typescript
// apps/web/src/components/country-selector.tsx → ALL_COUNTRIES
{ code: "XX", name: "Example", flag: "🇽🇽" },
```

## Currency Formatting

### Rust
```rust
lazynext_international::format::format_currency(amount_minor, "USD", "en-US") // "$12.99"
lazynext_international::format::format_currency(123456, "EUR", "fr-FR")       // "1.234,56 €"
```

### JavaScript
```typescript
import { formatCurrency } from "@/utils/format-currency";
formatCurrency(12.99, "USD", "en-US")  // "$12.99"
formatCurrency(1234.56, "EUR", "fr-FR") // "1 234,56 €"
```

## RTL Support
Arabic (ar) automatically renders right-to-left in the web app. The `<html dir="rtl">` attribute is set based on locale detection.

## MCP Server Locale
Set `MCP_LOCALE=fr` environment variable to switch the MCP server's error messages and prompt responses to French (or any supported language code).
