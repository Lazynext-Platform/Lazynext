# Lazynext Time

Media time types and multi-camera synchronization for the NLE.

## Overview

Provides the foundational time primitives used throughout Lazynext: frame rates, timecodes, media timestamps, and multi-camera sync clocking.

## Modules

- **crdt** — CRDT-compatible timestamp types for distributed editing
- **multicam** — Multi-camera synchronization via timecode and audio waveform alignment
- **captions** — Subtitle/caption timing with SRT/VTT import/export and frame-accurate positioning
